# Open-Oriented-Hypothesis

閉じてない仮説の鍛冶場  
— 反証可能な仮説をGitで管理するオープン指向プラットフォーム

## 目的
国際法などの「ふわっとした法案」を、科学的思考で「反証可能な仮説」に変換して四半期ごとに検証する。

## 参加方法
1. `/templates` から好きなテンプレートをコピー
2. Forkして改善 or 新規仮説作成
3. PRで送ってください
